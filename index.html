What is Git?

Git is software for tracking changes in any set of files, usually used for coordinating work among programmers collaboratively developing source code during software development. Its goals include speed, data integrity, and support for distributed, non-linear workflows.

How to install?

To install Git, run the following command: sudo apt-get install git-all . Once the command output has completed, you can verify the installation by typing: git version .

Installing on Windows ... There are also a few ways to install Git on Windows. The most official build is available for download on the Git website.

Local Repository

Create a folder and add a file in it. Git init command will initialize it with git locally

        git remote add origin

        This command will connect your local Repository to online one.

        git add .

        This command will keep track of all files of current directory

        git commit -m "message" 

        This command will commit new changes with specific message

        git branch -M main

        This command will create branch main as by default branch is main

        git push -u origin main

        This command will create online branch main and push your local changes to online Repository
